## Execution Strategy
1. **Extract Prompts**: For each file, identify and extract individual research prompts.
2. **Execute Research**: Use Google Search and other tools to gather data for each prompt, providing detailed paragraph-length responses.
3. **Format Deliverables**: Create markdown files for each prompt's results.
4. **Organize Artifacts**: Save in `research-artifacts` using the specified folder naming and nesting rules (max 3 words, lowercase, dashes; max 10 subfolders per directory).
5. **Mark Completion**: Prepend `[c]` to filenames and folder name upon completion. Do not enumerate files with numbers.
6. **Finalize**: Move the completed folder to `executed-prompts`.
