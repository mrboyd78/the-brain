2. **Execute Research**: Use Google Search and Web Fetch to gather data for each prompt.
3. **Format Deliverables**: Create markdown files for each prompt's results.
4. **Organize Artifacts**: Save in `research-artifacts` using the specified folder naming and nesting rules.
5. **Mark Completion**: Prepend `[c]` to filenames and folder name upon completion.
6. **Finalize**: Move the completed folder to `executed-prompts`.
